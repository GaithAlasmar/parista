# دليل الشراب والنكهات

نظام النكهات في باريستا مبني على **تشكيلة شراب وصلصة Monin** — معيار الصناعة للاتساق والجودة والتنوع. يوثّق هذا الدليل كل نكهة متاحة وكيفية استخدامها وما يمكن دمجه معها.

---

## الكمية القياسية

| الصيغة | الكمية |
|---|---|
| **الشراب في المشروبات الساخنة** | 15–20 مل |
| **الشراب في المشروبات الباردة/المثلجة** | 15–20 مل (أضف إلى القاعدة قبل الثلج) |
| **الصلصة في المشروبات الساخنة** | 20–30 غرام |
| **الصلصة في المشروبات الممزوجة** | 25–35 غرام |

> **أضف الشراب دائماً إلى الكوب قبل الإسبريسو** — الإسبريسو الساخن يذيب الشراب ويدمجه فوراً.

## الشراب الكلاسيكي — التشكيلة الأساسية

### عائلة الحلو والفانيلا
| النكهة | الملف | الأفضل في |
|---|---|---|
| **فانيلا** | نظيف، متناسق، حلو عالمي | لاتيه، كابتشينو، لاتيه مثلج، فرابيه |
| **فانيلا فرنسية** | فانيلا أغنى، تشبه الكريم كراميل | لاتيه، شوكولاتة ساخنة |
| **توفي نت** | زبداني، مكسّرات كراميل | لاتيه، لاتيه مثلج |
| **عسل** | زهري، ذهبي، حلو طبيعي | فلات وايت، كورتادو، شاي |
| **قرفة** | بهار دافئ، حلو-خشبي | كابتشينو، تشاي، شوكولاتة ساخنة |

### عائلة الكراميل
| النكهة | الملف | الأفضل في |
|---|---|---|
| **كراميل** | زبداني، ذهبي، يرضي الجميع | لاتيه، كابتشينو، فرابيه، ماكياتو |
| **كراميل مملّح** | كراميل مع حافة معدنية | لاتيه، لاتيه مثلج، فرابيه |

### عائلة المكسرات
| النكهة | الملف | الأفضل في |
|---|---|---|
| **بندق** | محمّص، دافئ، ترابي مكسّرات | لاتيه، كابتشينو، موكا |
| **لوز** | خفيف، حلو قليلاً، زهري | لاتيه، كورتادو |
| **فستق** | رقيق، ترابي-حلو، أخضر شاحب | لاتيه، لاتيه مثلج (استخدم الصلصة) |
| **آيريش كريم** | كريمي، يشبه الويسكي، حلو | لاتيه، شوكولاتة ساخنة |

## بناء التوليفات المخصصة

يمكن دمج أي شراب — المفتاح هو التوازن. **استخدم 10 مل من كل منهما** عند دمج شرابين (إجمالي 20 مل حلاوة).

### توليفات شائعة ومثبتة

| التوليفة | الاسم | المشروب الأساسي |
|---|---|---|
| فانيلا + بندق | "فانيلا مكسّرات" | لاتيه، كابتشينو |
| بندق + شوكولاتة داكنة | "نوتيلا" | لاتيه، موكا |
| كراميل + بندق | "كراميل مكسّرات" | لاتيه، لاتيه مثلج |
| فانيلا + لافندر | "لافندر فانيلا" | لاتيه، لندن فوغ |
| كراميل + قرفة | "سنيكرودول" | لاتيه، كابتشينو |
| فستق + ورد | "الحديقة الفارسية" | لاتيه مثلج |
| كراميل مملّح + توفي نت | "توفي مملّح" | فرابيه، لاتيه مثلج |
| جوز الهند + كراميل | "ساموا" | لاتيه مثلج، فرابيه |
| فراولة + فانيلا | "كريم الفراولة" | فرابيه، ميلك شيك |
| مانجو + جوز الهند | "استوائي" | لاتيه مثلج، فرابيه |

|---|---|---|---|---|
| لاتيه | | 26+ شراب | | 26+ لاتيه ساخن |
| كابتشينو | | 26+ شراب | | 26+ كابتشينو |
| موكا | | 26+ شراب | | 26+ موكا |

> مع توليفات الشرابين المزدوجة، تتجاوز إجمالي التوليفات الممكنة **100 وصفة فريدة**.

## نصائح احترافية لعمل النكهات

- **تذوّق قبل التقديم.** إذا أضفت توليفة شراب جديدة، تذوّقها قبل إرسالها للزبون.
- **توازن الحلاوة:** الشراب الفواكهي (مانجو، توت العليق) يتناسب بشكل أفضل مع مشروبات الحليب الأخف. الكراميل والمكسرات تتناسب بشكل أفضل مع قواعد الإسبريسو الجريئة.
- **درجة الحرارة مهمة:** بعض النكهات (لافندر، ورد) أكثر رقةً ساخنة — قلّل إلى 10 مل في المشروبات الساخنة وحافظ على 15 مل في الباردة.
- **بيع الشراب بالارتقاء:** "هل تود إضافة نكهة للاتيه الخاص بك؟" — اذكر هذا دائماً. الشراب إضافة عالية الهامش مع جهد ضئيل.
